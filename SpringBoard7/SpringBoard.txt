프로젝트 개발 순서

[SpringMVC]
SQL(mapper) -> DAO/DAOImpl -> Service/ServiceImpl -> Controller -> View

[SpringBoard]
Controller -> View -> Controller -> Service/ServiceImpl -> DAO/DAOImpl -> SQL(mapper)

	* Controller 준비
	- 공통적인 URI경로 설정(컨트롤러를 구분 목적)
	- 각 기능별 URI를 설정(+ 호출방식 GET/POST)
	- 연결된 뷰페이지 처리(정보전달Model / 페이지이동 redirect)
	
	* Service준비
	- 컨트롤러와 DAO를 연결 기능을 설정
	
	* DAO 준비
	- DB 연결 정보 처리(root-context.xml)
	- Mysql에 데이터베이스/테이블 생성
	- SQL 구문(mapper) 준비
	
	* DB생성 

	CREATE TABLE `tbl_board` (
  `bno` int NOT NULL AUTO_INCREMENT COMMENT '게시판에 글번호를 설정\nAI (자동으로 숫자를 증가시켜가면서 설정)',
  `title` varchar(200) NOT NULL,
  `content` text,
  `writer` varchar(50) NOT NULL COMMENT '글쓴이',
  `regdate` timestamp NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '작성날짜',
  `view` int DEFAULT '0' COMMENT '글 조회수 (기본값은 0)',
  PRIMARY KEY (`bno`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_0900_ai_ci COMMENT='게시판 정보를 저장하는 테이블'
